# Coming AI 개인화 기능 PRD — 업데이트 (2026-07-23)

> 기존 PRD 대비 **개인화 4대 기능**의 구현·배포 현황을 반영한 업데이트 문서입니다.
> 대상 독자: 협력사 / 외부 이해관계자. 내부 구현 세부는 생략하고 동작 기준으로 기술합니다.
>
> 상태: **4개 기능 모두 프로토타입에 구현·배포 완료** (2026-07-23 기준)

---

## 요약

학생의 온보딩 선호와 강의실 행동 데이터를 하나의 프로필로 통합해,
**① 프롬프트 추천 → ② AI 피드백(영상 채점) → ③ 모델 자동 선택 → ④ 학습 경로 추천**이
하나의 학습 루프(생성 → 채점 → 약점 진단 → 다음 강의)로 이어지도록 했습니다.

---

## 1. 프롬프트 추천

**무엇**: 강의 구간별 예시 프롬프트를 회원마다 다른 순서로 추천한다.

- 온보딩에서 받은 스타일 선호(시드)와 강의실 행동(구간 체류, 생성 이력, 별점 등)을
  종합한 회원 프로필로 후보 프롬프트를 재정렬.
- 추천 근거를 화면에 노출: "맞춤"(개인 취향 반영) / "추천"(전체 인기) 라벨 구분.
- 별도 카드로 분리돼 있던 "회원님 맞춤" 추천을 **추천 프롬프트 패널로 통합**해
  한 곳에서 추천을 받도록 UX 정리 (7/23).
- 추천 프롬프트에 **강의명·구간 정보 표기**, 긴 프롬프트는 더보기/접기 (7/23).
- 추천 품질 피드백: 추천 프롬프트로 생성한 뒤 **별점(1~5)** 을 받게 하고,
  별점은 다시 추천 순위에 반영된다.

**개인화 원리(개요)**: `추천 점수 = α·개인 취향 + (1−α)·전체 인기`.
데이터가 없는 신규 회원은 인기순으로 시작하고, 행동 데이터가 쌓일수록
개인화 비중(α)이 자동으로 커진다 — 콜드스타트에서도 항상 합리적인 추천이 나간다.

## 2. AI 피드백 (영상 채점)

**무엇**: 학생이 생성한 영상을 AI가 **정량 점수 + 정성 코멘트**로 채점한다.

- 4개 축 가중 평가: 프롬프트 충실도 35% · 시각적 품질 25% · 모션 자연스러움 20% ·
  시간적 일관성 20% → 종합 점수(0~100)와 등급.
- 정성 피드백: 잘한 점 / 개선점 / 아티팩트 지적을 문장으로 제공.
- 채점 결과 카드 UX 정비 (7/23): 헤더에 등급·점수, **좌측 제출 영상(16:9 고정) +
  입력 프롬프트, 우측 축별 점수 바 + AI 코멘트**의 좌우 비교 레이아웃.
- 채점된 영상은 저장되어 관리 화면(인사이트)에서 다시 재생·확인 가능 (7/23).

## 3. 모델 자동 선택

**무엇**: 회원 프로필에 맞춰 **AI 모델·해상도·영상 길이를 자동 제안/적용**한다.

- 우선순위: ① 본인이 실제 성공시킨 생성 설정 → ② 온보딩 선호 기반 추정 →
  ③ 전체 수강생 인기 설정 (콜드스타트 폴백).
- 신규 회원은 강의실 입장 시 **자동 적용**, 이력이 있는 회원에겐
  "추천 설정 · 이 설정으로 맞추기" 카드로 **원클릭 적용** 제안.
- 추천 근거 문구를 함께 표기 (예: "직접 만든 영상의 80%를 이 설정으로 작업하셨어요").
- 강의실 화면 구성(섹션·스타일)에도 개인화를 적용 (7/23).

## 4. 학습 경로 추천

**무엇**: 채점 결과에서 **가장 약한 축**을 찾아, 이를 보완하는 커리큘럼 강의를
AI 튜터 채팅으로 추천한다 (7/23 신규).

- 매핑 예: 프롬프트 충실도/모션/일관성 취약 → 프롬프트 설계 클래스,
  시각적 품질 취약 → 스타일 제어 클래스.
- 추천 문구에 회원의 선호 스타일을 반영해 개인화된 권유로 표현하고,
  해당 강의로 바로 이동하는 링크 제공.
- 채점표 확인 직후 자연스러운 후속으로 노출 (생성 1건당 1회).
- 이로써 **생성 → 채점 → 약점 진단 → 보완 강의 → 재학습**의 개인화 루프가 완성.

## 수집·활용 데이터 (개요)

| 데이터 | 수집 시점 | 활용 |
|---|---|---|
| 온보딩 선호 (콘텐츠·스타일·목적·숙련도) | 첫 진입 (건너뛰기 가능) | 콜드스타트 프로필 |
| 구간 체류 시간·이동 | 강의 시청 중 | 관심 구간 → 프롬프트 추천 |
| 모델·파라미터 선택, 생성 이력(성공/실패) | 영상 생성 시 | 모델 자동 선택 |
| 입력 프롬프트·수정 정도·출처(추천/직접) | 생성 시 | 스타일 취향 학습, 추천 품질 측정 |
| 별점 평가 | 추천 프롬프트 사용 후 | 추천 랭킹 보정 |
| 채점 결과 (축별 점수·코멘트) | 영상 채점 시 | 학습 경로 추천 |

- 최근 행동에 가중(14일 반감기)해 취향 변화를 따라간다.
- 모든 지표는 관리자용 **개인화 인사이트 대시보드**에서 회원별로 확인 가능
  (프로필, 추천 예측, 활동 패턴, 채점 이력, 회원 비교).

## 기존 PRD 대비 변경 요약

1. 프롬프트 추천: 별도 맞춤 카드 → **예시 프롬프트 패널 통합**, 추천 근거 라벨·강의명 표기 추가.
2. AI 피드백: 채점 카드 **좌우 비교 레이아웃**으로 재설계, 채점 영상 저장·재생 추가.
3. 모델 자동 선택: 강의실 **설정·섹션·스타일 개인화 적용** 확대.
4. 학습 경로 추천: **신규 기능** — 약점 축 기반 보완 강의를 튜터 채팅으로 추천.
